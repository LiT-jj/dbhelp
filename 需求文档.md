# 数据库测试工具需求文档

## 一、项目概述

### 1.1 项目名称
**DatabaseTestTool** - 数据库测试工具

### 1.2 项目目标
开发一款支持多数据库类型的测试工具，实现数据生成、约束管理、数据比对和性能测试等功能，为数据库开发和测试人员提供便捷的测试环境。

### 1.3 目标用户
- 数据库开发人员
- QA测试工程师
- DBA数据库管理员
- 数据工程师

---

## 二、功能需求

### 2.1 基础功能模块

#### 2.1.1 数据库连接管理
- 支持多种数据库类型的连接配置
- 支持的数据库类型：
  - MySQL
  - Oracle
  - GoldenDB
  - PostgreSQL
  - 其他主流数据库（可扩展）
- 提供连接信息的加密存储
- 支持连接测试功能
- 支持连接池管理

#### 2.1.2 数据库类型选择
- 下拉选择目标数据库类型
- 根据数据库类型动态加载对应的驱动
- 自动适配不同数据库的SQL语法

---

### 2.2 数据生成模块

#### 2.2.1 表结构解析
- 自动读取数据库表结构
- 解析字段信息（字段名、类型、长度、精度）
- 解析表注释和字段注释
- 支持DDL语句导入

#### 2.2.2 约束解析
- 主键约束（Primary Key）
- 外键约束（Foreign Key）
- 唯一约束（Unique）
- 非空约束（NOT NULL）
- 检查约束（CHECK）
- 默认值约束（DEFAULT）
- 自增约束（AUTO_INCREMENT/SERIAL）

#### 2.2.3 约束添加
- 提供图形化界面添加约束
- 支持创建主键（单字段/复合主键）
- 支持创建外键关系
- 支持创建唯一索引
- 支持添加检查约束
- 支持设置字段默认值
- 支持修改字段非空属性

#### 2.2.4 自定义约束生成
- **根据查询语句生成约束**：
  - 解析查询语句中的条件表达式
  - 识别字段与值的关系
  - 自动转换为约束规则
- **支持的约束类型**：
  - **范围约束**：基于查询中的范围条件（如 BETWEEN、>、<、>=、<=）
  - **等值约束**：基于查询中的等值条件（如 =、IN、LIKE）
  - **关联约束**：基于查询中的关联条件（如 JOIN 条件，可视为外键约束的扩展）
- **约束生成流程**：
  - 输入查询语句
  - 解析 SQL 语法
  - 提取约束条件
  - 生成约束规则
  - 应用到数据生成中
- **约束管理**：
  - 支持保存和加载自定义约束
  - 支持修改和删除已生成的约束
  - 支持约束优先级设置

#### 2.2.5 智能数据生成
- 根据字段类型自动生成数据：
  - 整数类型：自增序列、随机数、指定范围
  - 字符串类型：随机字符、UUID、姓名、地址、邮箱等模板
  - 日期时间类型：当前时间、随机日期、指定范围日期
  - 浮点类型：随机浮点数、指定精度
  - 布尔类型：随机布尔值、true/false比例控制
  - 大文本类型：随机段落、Lorem ipsum
  - 枚举类型：从预设值中随机选择
- 根据约束规则生成数据：
  - 主键字段：唯一自增或UUID
  - 外键字段：引用表中的有效值
  - 唯一字段：确保唯一性
  - 非空字段：确保不生成空值
  - 检查约束：根据约束条件生成合法数据
  - **自定义约束**：
    - 范围约束：在指定范围内生成数据
    - 等值约束：生成符合指定值或模式的数据
    - 关联约束：生成与关联表匹配的数据
- 支持批量生成指定数量数据
- 支持数据预览
- 支持生成SQL INSERT语句或直接插入

---

### 2.3 数据比对模块

#### 2.3.1 数据库选择
- 支持同时选择两个数据库进行比对
- 支持同类型数据库比对
- 支持异构数据库比对（如MySQL vs Oracle）

#### 2.3.2 数据同步插入
- 在两个数据库中插入相同的测试数据
- 自动处理不同数据库的SQL语法差异
- 确保插入数据完全一致
- 支持批量插入操作
- 插入结果验证

#### 2.3.3 查询结果比对
- 支持自定义查询语句
- 比对两个数据库的查询结果：
  - 结果集内容比对
  - 结果集行数比对
  - 结果集列结构比对
  - NULL值处理一致性
  - 浮点数精度容差比对
- 高亮显示差异内容
- 生成差异报告

#### 2.3.4 性能比对
- 记录查询执行时间
- 支持多次执行取平均值
- 支持多次执行取最小/最大值
- 性能指标展示：
  - 执行时长（毫秒）
  - 结果集大小
  - 吞吐量（行/秒）
- 生成性能对比报告

---

### 2.4 辅助功能

#### 2.4.1 SQL执行
- SQL语句编辑器（支持语法高亮）
- SQL格式化
- 执行计划展示
- 历史执行记录

#### 2.4.2 数据导出
- 导出为CSV格式
- 导出为JSON格式
- 导出为SQL脚本
- 导出比对报告

#### 2.4.3 项目管理
- 保存连接配置
- 保存数据生成模板
- 保存常用查询语句
- 导出/导入项目配置

---

## 三、非功能需求

### 3.1 性能需求
- 界面响应时间 < 1秒
- 单次数据生成操作 < 5秒（万级数据量）
- 查询比对操作响应及时，支持进度显示

### 3.2 兼容性需求
- 支持Windows、Linux、macOS操作系统
- 支持JDK 1.8及以上版本运行
- 支持主流浏览器访问（Web版本）

### 3.3 安全需求
- 数据库密码加密存储
- SQL注入防护
- 敏感信息脱敏展示

### 3.4 可扩展性需求
- 插件式数据库驱动管理
- 支持自定义数据生成规则
- 支持自定义比对规则

---

## 四、技术选型

### 4.1 桌面应用版本
- **前端**：JavaFX / Swing
- **后端**：Java
- **数据库驱动**：JDBC

### 4.2 Web应用版本
- **前端**：Vue.js / React
- **后端**：Spring Boot
- **数据库驱动**：JDBC / 各数据库原生驱动

## 五、界面原型（简要）

### 5.1 主界面布局
```
┌─────────────────────────────────────────────────────┐
│  菜单栏（文件、编辑、视图、工具、帮助）               │
├──────────┬──────────────────────────────────────────┤
│          │                                          │
│  导航栏   │           主工作区                       │
│  - 连接   │                                          │
│  - 表结构 │                                          │
│  - 造数   │                                          │
│  - 自定义约束 │                                      │
│  - 比对   │                                          │
│  - 设置   │                                          │
│          │                                          │
├──────────┴──────────────────────────────────────────┤
│  状态栏（连接状态、当前操作、进度）                   │
└─────────────────────────────────────────────────────┘
```

### 5.2 核心页面
1. **连接管理页面**：数据库连接列表、添加/编辑/删除连接、连接测试
2. **表结构页面**：表列表、字段列表、约束列表、DDL预览
3. **数据生成页面**：表选择、字段配置、生成规则、预览/执行
4. **自定义约束页面**：查询语句输入、约束解析、约束管理、应用到数据生成
5. **数据比对页面**：数据库选择、查询输入、结果展示、性能对比
6. **设置页面**：系统设置、驱动管理、模板管理

## 七、风险评估

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| 不同数据库SQL语法差异 | 高 | 抽象SQL层，针对性适配 |
| 大量数据生成性能 | 中 | 分批处理，显示进度 |
| 多数据库驱动兼容性 | 中 | 使用标准JDBC接口测试 |
| 约束解析准确性 | 中 | 充分测试各类约束场景 |
| SQL查询解析复杂性 | 中 | 使用成熟的SQL解析库，增加测试用例 |
| 自定义约束冲突 | 低 | 实现约束优先级机制，提供冲突检测 |
