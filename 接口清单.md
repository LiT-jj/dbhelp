# 接口清单

## 连接管理
- `GET /api/connections`：获取连接列表
- `POST /api/connections`：新增连接
- `PUT /api/connections/{id}`：更新连接
- `DELETE /api/connections/{id}`：删除连接
- `POST /api/connections/test`：测试连接
- `GET /api/connections/export`：导出连接配置
- `POST /api/connections/import`：导入连接配置（`multipart/form-data`）

## 表结构
- `GET /api/tables`：获取表列表
- `GET /api/tables/{tableName}/columns`：获取字段
- `GET /api/tables/{tableName}/constraints`：获取约束
- `POST /api/tables/ddl`：解析 DDL
- `GET /api/tables/{tableName}/ddl`：获取表 DDL

## 数据生成
- `POST /api/generate/data`：执行生成
- `POST /api/generate/sql`：生成 SQL
- `POST /api/generate/preview`：预览生成数据
- `GET /api/generate/progress?taskId=xxx`：查询进度

## 自定义约束
- `GET /api/constraints`：获取约束列表
- `POST /api/constraints`：新增约束
- `PUT /api/constraints/{id}`：更新约束
- `DELETE /api/constraints/{id}`：删除约束
- `POST /api/constraints/parse`：解析 SQL 约束
- `POST /api/constraints/{id}/apply`：应用约束

## 数据比对
- `POST /api/compare/execute`：执行比对
- `POST /api/compare/sync`：数据同步
- `POST /api/compare/export`：导出比对报告
- `GET /api/compare/progress?taskId=xxx`：查询比对进度

## 辅助功能
- `POST /api/sql/execute`：执行 SQL
- `POST /api/sql/format`：格式化 SQL
- `GET /api/sql/history`：SQL 历史
- `POST /api/export/data`：导出表数据
- `POST /api/export/result`：导出查询结果

## 项目管理
- `GET /api/projects`：项目列表
- `POST /api/projects`：创建项目
- `PUT /api/projects/{id}`：更新项目
- `DELETE /api/projects/{id}`：删除项目
- `GET /api/projects/{id}/export`：导出项目
- `POST /api/projects/import`：导入项目
